📖 만든 이의 이야기📖 Maker's note

반갑습니다. FLOW: AX 디자인연구소 임정훈 소장입니다. Windows 용 FLOW ZoomIt 을 만들고 나서 "맥북용도 만들어 주세요" 라는 요청을 여러 차례 받았습니다.

Microsoft 는 ZoomIt 을 macOS 로 포팅하지 않았기 때문에, Mac 강사·코치는 동급 도구가 없었습니다. 마침 @07JP27 님의 오픈소스 ZoomacIt (Swift 6 + AppKit 네이티브) 이 ZoomIt 의 정신적 클론으로 활발히 개발 중이어서, GPL-3.0 라이선스로 포크해 한국어 강의 디폴트를 입히는 방향으로 진행하고 있습니다.

Windows 버전 개발 과정에서 Electron + 자체 IME 로 5번 시도했다가 모두 실패했던 경험이 있어, Mac 에서도 "Day 1 한글 IME 검증" 을 먼저 통과한 뒤 본격 빌드에 들어갑니다. 시간이 좀 걸리더라도 한국어 입력이 정상 작동하는 빌드만 내놓으려고 합니다.

많은 격려 감사합니다. 진행 상황은 GitHub 저장소에서 공유드릴게요. Hi, I'm Junghoon Lim, Director of FLOW: AX Design Lab. After releasing the Windows-only FLOW ZoomIt, multiple Korean trainers asked for a Mac equivalent.

Microsoft never ported ZoomIt to macOS, so Mac-using lecturers and coaches had no direct counterpart. @07JP27's open-source ZoomacIt (Swift 6 + native AppKit) is a spiritual clone of ZoomIt and actively developed, so I'm forking it under GPL-3.0 and layering FLOW Korean defaults on top.

During the Windows build I tried five Electron-based prototypes with custom Korean IMEs — all failed because transparent Chromium windows are incompatible with Hangul composition. So for the Mac fork I'm deliberately gating on a Day-1 Korean IME verification before any further work. A working Korean input is non-negotiable.

Thank you for the encouragement. I'll share progress here on the repo.

왜 만들었나Why this exists

Microsoft Sysinternals ZoomIt 은 Windows 전용입니다. macOS 사용자는 한국어 강의용 화면 강조 도구를 직접 찾아야 했습니다.

마침 @07JP27 의 오픈소스 ZoomacIt (Swift 6 + AppKit 네이티브) 이 ZoomIt 의 정신적 클론으로 활발히 개발 중입니다 (v0.5.0, 2026-03). 이를 GPL-3.0 라이선스 하에 포크해 한국어 강의 디폴트(손글씨 폰트 fallback chain·6색 펜 팔레트·B/W 안전 가이드·양국어 문서)를 입혀 FLOW ZoomIt for Mac 을 만들었습니다. Microsoft never ported ZoomIt to macOS. Korean trainers on Mac had no equivalent.

@07JP27's open-source ZoomacIt (Swift 6 + AppKit, native macOS) is a spiritual clone of Windows ZoomIt and actively developed (v0.5.0, March 2026). Forked under GPL-3.0 with Korean classroom defaults — handwriting-font fallback chain, full pen palette, B/W safety guidance, bilingual docs.

⚠️ B/W 캔버스 덮어쓰기 주의 (Windows 와 동일)⚠️ B/W canvas-blank gotcha (same as Windows)

그리기 모드(⌃2)에서 T 를 누르기 전에 한국어 IME 로 바꾸면, 첫 자모 ㅈ(=w) / ㅠ(=b) 가 캔버스 전체를 덮어버립니다. In Draw mode (⌃2), pressing standalone B/W blanks the canvas. If Korean IME is on before pressing T, the first Hangul jamo (ㅈ=w / ㅠ=b) will trigger this.
