# Keyboard Shortcuts

A complete reference of all keyboard shortcuts in ZoomacIt, organized by feature.

## Global Hotkeys

| Shortcut | Action |
|---|---|
| **⌃1** (Control+1) | Enter Zoom mode |
| **⌃2** (Control+2) | Enter Draw mode |
| **⌃3** (Control+3) | Start Break Timer |

::: tip
These hotkeys can be customized in **Settings**.
:::

## Zoom Mode

| Input | Action |
|---|---|
| Mouse move | Pan |
| Scroll wheel | Zoom in / out |
| ↑ | Zoom in |
| ↓ | Zoom out |
| Click | Enter Draw mode |
| Escape | Exit Zoom mode |
| Right-click | Exit Zoom mode |

## Draw Mode

### Drawing

| Input | Action |
|---|---|
| Drag | Freehand drawing |
| Shift + Drag | Straight line |
| Control + Drag | Rectangle |
| Tab + Drag | Ellipse |
| Shift + Control + Drag | Arrow |

### Colors

| Key | Color |
|---|---|
| R | Red (default) |
| G | Green |
| B | Blue |
| O | Orange |
| Y | Yellow |
| P | Pink |
| Shift + color key | Highlighter mode |

### Tools

| Key | Action |
|---|---|
| T | Enter text input mode |
| ⌃ + scroll wheel | Change pen width |
| E | Erase all drawings |
| W | Whiteboard background |
| K | Blackboard background |

### Actions

| Key | Action |
|---|---|
| ⌘Z | Undo last stroke |
| ⌘C | Copy canvas to clipboard |
| ⌘S | Save canvas to file |
| Space | Center cursor on screen |
| Escape | Exit text mode / Exit Draw mode |
| Right-click | Exit Draw mode |

### Text Mode

| Input | Action |
|---|---|
| Click | Place text field |
| Scroll wheel | Change font size |
| R / G / B / O / Y / P | Change text color |
| Escape | Confirm text, return to pen mode |
| Right-click | Confirm text, exit Draw mode |

## Break Timer

| Input | Action |
|---|---|
| ↑ | Add 1 minute |
| ↓ | Subtract 1 minute |
| R / G / B / O / Y / P | Change timer text color |
| Escape | Dismiss timer |
