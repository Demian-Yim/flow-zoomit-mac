# キーボードショートカット

ZoomacIt の全キーボードショートカットの一覧です。機能別にまとめています。

## グローバルホットキー

| ショートカット | アクション |
|---|---|
| **⌃1**（Control+1） | ズームモードに入る |
| **⌃2**（Control+2） | ドローモードに入る |
| **⌃3**（Control+3） | 休憩タイマーを開始 |

::: tip
これらのホットキーは**設定**でカスタマイズできます。
:::

## ズームモード

| 入力 | アクション |
|---|---|
| マウス移動 | パン |
| スクロールホイール | ズームイン / ズームアウト |
| ↑ | ズームイン |
| ↓ | ズームアウト |
| クリック | ドローモードに入る |
| Escape | ズームモードを終了 |
| 右クリック | ズームモードを終了 |

## ドローモード

### 描画

| 入力 | アクション |
|---|---|
| ドラッグ | フリーハンド描画 |
| Shift + ドラッグ | 直線 |
| Control + ドラッグ | 矩形 |
| Tab + ドラッグ | 楕円 |
| Shift + Control + ドラッグ | 矢印 |

### 色

| キー | 色 |
|---|---|
| R | 赤（デフォルト） |
| G | 緑 |
| B | 青 |
| O | オレンジ |
| Y | 黄 |
| P | ピンク |
| Shift + 色キー | 蛍光ペンモード |

### ツール

| キー | アクション |
|---|---|
| T | テキスト入力モードに入る |
| ⌃ + スクロールホイール | ペン幅の変更 |
| E | すべての描画を消去 |
| W | ホワイトボード背景 |
| K | ブラックボード背景 |

### アクション

| キー | アクション |
|---|---|
| ⌘Z | 最後のストロークを元に戻す |
| ⌘C | キャンバスをクリップボードにコピー |
| ⌘S | キャンバスをファイルに保存 |
| Space | カーソルを画面中央に移動 |
| Escape | テキストモード終了 / ドローモード終了 |
| 右クリック | ドローモード終了 |

### テキストモード

| 入力 | アクション |
|---|---|
| クリック | テキストフィールドを配置 |
| スクロールホイール | フォントサイズ変更 |
| R / G / B / O / Y / P | テキストの色を変更 |
| Escape | テキストを確定し、ペンモードに戻る |
| 右クリック | テキストを確定し、ドローモードを終了 |

## 休憩タイマー

| 入力 | アクション |
|---|---|
| ↑ | 1分追加 |
| ↓ | 1分減少 |
| R / G / B / O / Y / P | タイマーの文字色を変更 |
| Escape | タイマーを終了 |
